# 📂 Project Structure

This repository contains beginner-friendly Python programs that demonstrate the core concepts of programming. Each file focuses on a specific topic, making it easy to learn and practice Python fundamentals.

```text
PROGRAMMING-FUNDAMENTALS/
│
├── screenshots/            # Images used in the README documentation
├── comments.py             # Demonstrates how to write comments in Python
├── control_structures.py   # Examples of if, elif, and else statements
├── dictionary.py           # Working with Python dictionaries
├── forloops.py             # Examples of for loops and iteration
├── hello_world.py          # First Python program that prints output
├── indentation.py          # Demonstrates Python's indentation rules
├── lists.py                # Creating and manipulating lists
├── README.md               # Project documentation
├── strings_con.py          # String concatenation examples
├── strings.py              # String operations and methods
├── variables.py            # Variable declaration and data types
└── while_loops.py          # Examples of while loops
```

# 📘 File Descriptions

| File                      | Purpose                                                                 |
| ------------------------- | ----------------------------------------------------------------------- |
| hello_world.py        | Introduces Python by printing text to the console.                      |
| variables.py          | Demonstrates how to create variables and store different types of data. |
| comments.py           | Shows how comments improve code readability and documentation.          |
| indentation.py        | Explains Python's indentation requirements for code blocks.             |
| control_structures.py | Demonstrates decision-making using `if`, `elif`, and `else` statements. |
| forloops.py           | Shows how to repeat tasks using `for` loops.                            |
| while_loops.py        | Demonstrates repetition using `while` loops.                            |
| lists.py              | Introduces Python lists, indexing, and list manipulation.               |
| dictionary.py         | Explains how to store data using key-value pairs.                       |
| strings.py            | Covers string creation and common string methods.                       |
| strings_con.py        | Demonstrates combining multiple strings using concatenation.            |
| screenshots/          | Contains screenshots documenting project execution and results.         |
| README.md             | Provides an overview, setup instructions, and project documentation.    |

# 🎯 Learning Objectives

This project demonstrates the following Python programming fundamentals:

* Writing and executing Python programs
* Understanding variables and data types
* Using comments for code documentation
* Following proper Python indentation
* Making decisions with conditional statements
* Repeating tasks using `for` and `while` loops
* Working with lists and dictionaries
* Manipulating and combining strings
* Organizing a Python project using a clear file structure
